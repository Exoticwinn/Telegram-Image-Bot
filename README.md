# Telegram Bot with Image Recognition and Interactive Features

Многофункциональный Telegram‑бот с распознаванием изображений через нейросеть и интерактивными командами.

## Функциональность

Бот поддерживает следующие возможности:

* **Распознавание изображений** — анализирует фото с помощью обученной Keras‑модели и возвращает название объекта с вероятностью.
* **Экологический квиз** — серия из 5 вопросов о времени разложения разных материалов (пластик, стекло и т. д.).
* **Развлекательные команды** — мемы, случайные фото собак, эмодзи и др.
* **Утилитарные функции** — генерация паролей и случайных чисел, обратный отсчёт, подбрасывание монеты.
* **Информационный контент** — случайные факты и данные об экологических организациях.
* **Эхо‑режим** — повторяет текстовые сообщения пользователя.

## Установка и запуск

### Требования

* Python 3.8+
* Библиотеки:
  * `pyTelegramBotAPI`
  * `tensorflow` / `tf-keras`
  * `Pillow`
  * `numpy`
  * `requests`

pip install pyTelegramBotAPI tensorflow pillow numpy requests
