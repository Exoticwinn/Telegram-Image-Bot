# Telegram Bot with Image Recognition and Interactive Features


Многофункциональный Telegram‑бот с распознаванием изображений через нейросеть и набором интерактивных команд. Проект с открытым исходным кодом (open‑source).

## Описание

Бот объединяет возможности машинного обучения и интерактивного взаимодействия с пользователем.

**Основные функции:**

* **Распознавание изображений** — бот анализирует присланное фото с помощью обученной модели Keras (`keras_model.h5`) и определяет, что на нём изображено. Модель ожидает изображение размером 224×224 пикселя, RGB‑формат. Результат включает название класса (из файла `labels.txt`) и уровень уверенности (вероятность).
* **Экологический квиз** — серия из 5 вопросов о времени разложения различных материалов (пластиковая бутылка, алюминиевая банка, стеклянная бутылка и т. д.). Бот отслеживает счёт и показывает итоговый результат.
* **Развлекательные и утилитарные команды** — генерация паролей, случайных чисел, обратный отсчёт, подбрасывание монеты, отправка мемов и случайных фото собак.
* **Информационный контент** — случайные факты и данные об экологических организациях.
* **Эхо‑режим** — повторяет текстовые сообщения пользователя.

## Установка и запуск

### Требования

* Python 3.8+
* Библиотеки:
  * `pyTelegramBotAPI`
  * `tensorflow` / `tf-keras`
  * `Pillow`
  * `numpy`
  * `requests`

### Установка зависимостей

```bash
pip install pyTelegramBotAPI tensorflow pillow numpy requests
